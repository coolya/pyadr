============
Installation
============


Stable release
--------------

To install ADR Process Tool, run this command in your terminal:

.. code-block:: console

    $ pip install pyadr

This is the preferred method to install ADR Process Tool, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for ADR Process Tool can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/esciara/pyadr

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://gitlab.com/opinionated-digital-center/pyadr/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://gitlab.com/opinionated-digital-center/pyadr
.. _tarball: https://gitlab.com/opinionated-digital-center/pyadr/tarball/master
